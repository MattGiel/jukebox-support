How To

Step-by-step guides for JukeBox.

Create a Room

  1. Open JukeBox → Create Room.
  2. Enter a Room Name (for example, Matt’s Room) and confirm.
  3. That device controls playback and Song Host rotation.

Share the Room Name (no invites required)

Tell friends the exact Room Name so they can search for it.

  • Make sure they have JukeBox installed and open.
  • Share the spelling and spacing exactly as you created it.

Join a Room by Name

  1. Open JukeBox → Join Room.
  2. Search for the Room Name you were given.
  3. Select the room and confirm to join.

Taking Turns (“Song Host”)

The Song Host rotates so everyone gets a turn. If you set a designated playlist, JukeBox pulls your next track in order when it’s your turn.
